

### Pre-setup
Download wasm-pack
`https://rustwasm.github.io/wasm-pack/installer/`

Rust wasm32
`rustup target add wasm32-unknown-unknown --toolchain nightly`


## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
```
